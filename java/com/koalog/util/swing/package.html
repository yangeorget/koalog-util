Provide a set of classes that let define an application from a XML file.

The global structure of the XML file is as follows:

  • action declarations: each of the application actions are declared using an <ACTION> element;
  • menu declarations: the menus are declared using <MENU> elements, and menu items are declared with <MENUITEM> elements, which are references to the declared actions.

Application definition XML files can be parsed using the ApplicationLoader class.

In order to be loaded using an ApplicationLoader instance, an application must implement the AutoApplication interface.
